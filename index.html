Minecraft Note Block Music — FAQ

How do note blocks work in Minecraft?

Right-click a note block to raise its pitch by one semitone (25 total notes). The instrument is determined by the block placed directly underneath. Power with Redstone to play. Connect multiple note blocks with repeaters for timing control.

What are all 16 Minecraft note block instruments?

Harp (any block), Bass Guitar (wood planks), Bass Drum (stone), Snare (sand/gravel), Hi-hat (glass), Guitar (wool), Flute (clay), Bell (gold block), Chime (packed ice), Xylophone (bone block), Iron Xylophone (iron block), Cow Bell (soul sand), Didgeridoo (pumpkin), Bit/8-bit (emerald block), Banjo (hay bale), Pling (glowstone).

What block goes under a note block for piano sound?

Place any default block (dirt, stone, any block not on the specific list) under a note block for the Harp sound — the closest to piano. Use bone block for xylophone, gold block for bell, and packed ice for chime tones.

How do I build a song with note blocks in Minecraft?

1) Plan your melody using note numbers 1–25. 2) Place note blocks and right-click each to set pitch. 3) Connect with Redstone repeaters (adjust ticks for tempo). 4) Loop with a comparator clock. 5) Layer multiple instruments on parallel tracks. The Songs tab above has step-by-step patterns for 5 famous songs.

What is the note range of Minecraft note blocks?

Note blocks cover exactly 2 octaves — 25 notes from F#1 to F#3. Note 1 is the lowest (F#1), note 13 is middle F#2, and note 25 is the highest (F#3). Each right-click raises pitch by one semitone. To reset to lowest, break and replace the note block.

Minecraft Note Block Music Builder — Complete Instrument & Song Guide

Minecraft note blocks are one of the game's most powerful creative features — but most players never go beyond a single note or basic melody. This guide covers everything you need to build real music in Minecraft: all 16 instruments and the exact blocks that activate them, the complete 25-note pitch chart from F#1 to F#3, and step-by-step build patterns for five famous songs including Megalovania, Super Mario Bros, and Pirates of the Caribbean.

The key to Minecraft music is understanding Redstone timing. Each Redstone repeater tick equals 0.1 seconds. A standard melody runs at 2–3 ticks per note for an upbeat tempo, 4–5 ticks for a slower feel. Stacking multiple note block tracks in parallel lets you build chords, basslines, and drum rhythms simultaneously — creating full arrangements inside the game.

For complex builds, organize your tracks vertically: melody on top level, harmony in the middle, bass at ground level, and percussion as a separate horizontal track. Use different instruments on each layer for a full orchestral sound.
